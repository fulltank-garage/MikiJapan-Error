# MikiJapan Error

หน้า Error แบบ frontend-only สำหรับแสดงโลโก้ร้านและข้อความให้ติดต่อผู้ดูแลระบบหากข้อมูลผิดพลาด

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS ผ่าน `@tailwindcss/vite`

## Run

Use Node.js 24.x.

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

โปรเจกต์นี้ไม่มีการเชื่อมต่อ backend, API หรือ LIFF
